About This Project

High-level description, learning goals, and technology stack used in the Disk Scheduling Simulator.

Project Description

The Advanced Disk Scheduling Simulator is a teaching tool for understanding how operating system disk scheduling algorithms impact performance. It focuses on four classical algorithms: FCFS, SSTF, SCAN and C-SCAN.

Instead of memorizing formulas and static examples, students can experiment by entering their own request patterns, observing head movement and comparing the total seek time, average seek time and throughput for each algorithm.

Key Features

  • Interactive, browser-based simulator implemented using HTML, CSS and JavaScript.
  • Visualization of head movement over the disk cylinder axis.
  • Support for FCFS, SSTF, SCAN and C-SCAN algorithms.
  • Automatic calculation of total seek time, average seek time and throughput.
  • Automatic detection of the best algorithm (minimum total seek time).
  • Random workload generator for quick testing and experimentation.
  • Export of metrics to CSV and PDF formats for lab reports and documentation.
  • Soft mint-green theme with a clean, modern interface.

Technology Stack

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Visualization: Chart.js for real-time line charts
  • PDF Export: jsPDF for generating metric reports
  • CSV Export: Browser Blob API and dynamic link download

The project runs fully in the browser with no backend server required. This makes the simulator easy to distribute and run on any system that has a modern web browser.
